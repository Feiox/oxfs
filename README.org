* Oxfs

  - Oxfs is a network file system like sshfs.
  - It is very fast to edit remote files with desktop software.

** Get start

   #+begin_src bash
   $ pip install oxfs
   $ mkdir remote
   $ sudo oxfs -s user@xxx.xxx.xxx.xxx -m remote -r /home/oxfs -p /tmp/oxfs
   #+end_src
