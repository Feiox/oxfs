* oxfs

  - A simple sftp filesystem with powerfull cache.

** mount

   #+begin_src bash
   $ mkdir remote
   $ oxfs -s user@xxx.xxx.xxx.xxx -m remote -p /tmp/oxfs
   #+end_src
